# 티츄 게임 아이디어 노트

    1. 게임을 접속하면 인트로 이후 메인 화면으로 이동함.
    2. 메인 화면에는 게임 시작, 게임 방법, 상점, 랭킹, 게임 종료 버튼이 존재하고,
    3. 게임 시작 버튼을 누르면 랭킹전, AI전, 친구와 게임하기 3가지의 버튼이 존재함.
    4. BM을 위해 가끔 나타나는 광고가 있어야할까?(보류)
    5. 상점은 유저를 꾸밀 수 있는 아바타와 광고제거와 같은 아이템을 판매함.(근데 아바타는 모델링이 많이 필요해서 아이템은 일단 보류)
