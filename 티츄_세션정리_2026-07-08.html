2026-07-06 ~ 07-08 · Phase 2 AI 강화 + 플레이테스트 픽스 · main b04e09c = origin 완전 동기화
한 줄. 이번 세션은 잔여 AI 레버 소진(C1·D2·C3)과 사용자 3회 플레이테스트 피드백 반영이 핵심이었다. 총 10건 채택 · 5건 파킹 · 2건 한계 수용. 모든 강도 변경은 격리벤치로 판정했고, 모든 채택분은 origin에 반영됐다. 아직 완벽하진 않지만 관측된 나쁜 수의 대부분이 교정됐다. 다음은 실기 플레이테스트부터.

13에이전트 적대검증으로 잔여 6레버를 판정 → 값싼 레버는 D2로 소진. C3는 근사(opponent 강도 추측)라 최선케이스서도 ~0 → 파킹. 나머지(C2·D3·A1·A3)는 회귀/wash 예측으로 미착수.

3. 플레이테스트 2·3차 — 콜러 낭비 · 엔드게임

관측벤치판정
콜러 밟기 콜러가 파트너 A를 용으로 밟음+0.9/R 중립채택
Issue 1 교환서 파트너 티츄콜 시 탑패(용/봉황) 헌납0.0/R 중립채택
Issue 2 near-out 상대에 낮은 싱글 리드 금지 → 최고 싱글+0.1/R 중립채택
Issue A 콜러 고콤보/봉황 낭비 밟기 가드풀링 +0.18 · 승 동률채택
Issue 3 엔드게임 리드 순서 (1,9,A → A 먼저)셰딩가드 재벤치 −10.18한계
Issue C 팀이 개/저싱글로 콜러 지원개 Rank 0 = 최저이미 구현

Issue C는 코드로 검증 — 개(Dog) Strength가 최저라 AI가 이미 개/낮은 싱글로 콜러에 선을 넘긴다. 실제 문제는 "팀 미지원"이 아니라 "콜러의 자기 낭비"(콜러밟기·Issue A로 수정).

4. 파킹·한계 — 왜 안 고쳤나 (정직한 부분)

  • Issue 3 · ② 라지티츄 응징 = 롤아웃 천장. 다중 트릭 재진입 계획이나 콜러 아웃확률 모델링은 휴리스틱·라이브 EV가 못 한다. 엔드게임 셰딩가드 재벤치도 −10.18/R로 회귀 확인 → 리드 오버라이드 5번째 회귀. 진짜 해법은 연구티어(롤아웃 강화 / B1 α-μ).
  • ⑥ 교환 봉황 양도 · ⑧ 폭탄 세이브 · C3 = 벤치가 이득 없음/회귀를 정직히 걸러냄. 코드는 보존(플래그 OFF), 봉황 양도는 불변식 위반이라 폐기.
  • 핵심 교훈 재확인: 인간 눈에 "낭비"로 보이는 수 상당수가 실제 EV상 옳다. 그래서 모든 강도 변경은 격리벤치로 판정했고, 중립·재현교정만 채택했다.
